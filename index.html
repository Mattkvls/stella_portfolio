Text Classification using String Kernels

In this project the String Subsequence Kernel (SSK) algorithm is re-implemented. SSK is a kernel method introduced by Lodhi et al., 2002. A Kernel is a function that projects two data points into a higher dimensional space where the points can be linearly separable. It can be considered as a representation of similarity, and a mechanism by which domain knowledge id injected into the SVM algorithm. The Kernels can define a feature space for Support Vector Machines algorithm, a linear classifier used for the purposes of categorizing text documents. The results yielded are assessed by comparing with the results of 2002, Lodhi et al. A comparison was also conducted between the efficiency of SSK and n-Grams (NGK), Word Kernel (WK) as well as Latent Semantic Kernel and SIF model which is a sentence classifier based on a weighted mean of GloVe vectors. Experimental comparisons of the methods mentioned, showed that Latent Semantic Kernel has the best performance.

Reproducibility Project (contributor)

The aim of this project was to re-implement methods discussed in the paper "Learning where to look: Semantic-Guided Multi-Attention Localization for Zero-Shot Learning"[1], a paper accepted to NeurIPS2019. The objective of this paper is to classify instances by semantic-guided Zero Shot Learning to seen and unseen data, enhanced by focusing on discriminative parts of the images using attention.  A baseline to the original model is also introduced, which is only includes the original image’s global features and does not focus on any discriminative parts. All models were reproduced from scratch using Tensorflow 2.0.

Adversarial Attacks

Small changes called adversarial attacks that are not noticeable for humans can make inputs completely unrecognizable for deep neural networks, similar to optical illusions. In this project two methods for white-box adversarial attacks are tested: FGSM and PGD. Black-box generalizations of the attacks are tested by running them cross-model between two classification algorithms. Adversarial attacks are transferable between networks and can be applied in real-world settings. This is a problem for neural networks in safety-critical environments as well as an illustration of a lack of generality.
